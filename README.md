# TimeBridge Scheduler

A small static scheduler for choosing a biweekly fixed weekend meeting time.

- Saturday and Sunday only
- 9:00 AM to 12:00 AM
- Anchored to Philadelphia time (`America/New_York`)
- Timezone-aware display for participants

Public invite path:

```text
https://YOUR-GITHUB-PAGES-URL/#weekend
```

This is a static app. Availability is stored in the share link itself, so after
someone adds availability they should copy and send the updated link.
